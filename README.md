# 🧟 Resident Slug: Survival Horror 2D

![Resident Slug Banner](https://img.shields.io/badge/Game-Survival--Horror-red?style=for-the-badge)
![Tech](https://img.shields.io/badge/Engine-Canvas--2D-blue?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Beta--V1.0-green?style=for-the-badge)

**Resident Slug** es un shooter de supervivencia lateral inspirado en clásicos como *Metal Slug* y *Resident Evil*. El juego combina acción frenética de disparos con mecánicas de gestión de recursos, ambientado en una atmósfera de horror pixel-art.

Desarrollado íntegramente en **JavaScript puro (Vanilla JS)** utilizando la API de Canvas 2D.

---

## 🎮 Características Principales

- **Sistema de Combate Avanzado:** - Mira láser dinámica que cambia de color al detectar enemigos.
  - Gestión de munición (Cargadores y reserva).
  - Sistema de partículas de sangre y "muzzle flash" en cada impacto.
- **Movimiento Fluido:** Salto, agachado e inversión de sprites según la dirección.
- **Enemigos Variados:** Desde zombies básicos hasta perros mutantes y un jefe final con fases de combate.
- **UI Estilo Arcade:** Minimapa/Radar en tiempo real y contador de enemigos restantes inspirado en juegos retro.
- **Audio Dinámico:** Efectos de sonido sintetizados mediante *WebAudio API*.

## 🛠️ Tecnologías Utilizadas

- **Lenguaje:** JavaScript (ES6+)
- **Renderizado:** HTML5 Canvas API
- **Estilos:** CSS3 & Bootstrap 5
- **Audio:** WebAudio API (Osciladores y nodos de ganancia)
